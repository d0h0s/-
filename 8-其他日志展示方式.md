## log命令2  
如果历史记录有很多，查看日志时会出现分页效果，则会在当页的最下行出现：号。  
>下一页：空格  
上一页：b  
到尾页了，显示END  
推出：q  


#日志展示方式  
【1】方式1：git log ——>分页  
【2】方式2：  

    git log --pretty=oneline  
            （漂亮的）（显示在一行）  

【3】方式3：  

    git log --oneline  

会截取16进制码的前7位，达到简化的效果。  
【4】方式4：  

    git reflog  

多了HEAD@{数字}  
这个数字的含义：指针回到当前这个历史版本需要走多少步。
